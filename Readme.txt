README

게임 제목: [Strange Cabin]

조작법
화살표 키
위/아래 화살표: 계단이 있는 곳에서만 위아래로 이동 가능.
왼쪽/오른쪽 화살표: 좌우 이동 및 대시 취소.

스페이스바: 대시 (3초간 질주 가능, 왼쪽/오른쪽 화살표 키로 취소 가능).

C 키: 아이템 사용.
 	포션 사용 시 바라보는 방향으로 날라간다
	키 사용 시 철창이 열린다

엔터 키: 종이 더미나 촛불이 올라간 수납장 뒤에 숨기 
(은신 오브젝트 영역 안에서 누르고 있으면 몬스터가 감지하지 못함).

게임 진행
맵 탐험: 몬스터와 마주치지 않도록 주의하며 화살표 키를 사용해 맵을 탐험

맵 내의 포션을 사용하여 몬스터를 처치.
몬스터를 처치하면 다음 방으로 가는 문이 열림.

몬스터 심장 수집
몬스터를 처치할 때마다 하단 슬롯에 몬스터 심장이 하나씩 채워짐
.
모든 슬롯이 채워지면 거실을 지나 막힌 문이 열리고, 엔딩으로 이동

생존 가이드
숨기: 종이 더미나 촛불이 올라간 수납장 뒤에 숨어 엔터 키를 누르고 있으면 몬스터가 감지하지 못함
질주: 몬스터에게 잡히기 전에 빠르게 사용하면 탈출 가능함

몬스터 특징
특정 거리 이내로 접근 시 플레이어를 쫓아옴
평소에는 천천히 돌아다니다가 가끔 멈춰서 냄새를 맞는 행동을 취함
몬스터에게 잡히면 즉사

엔딩
모든 몬스터 심장을 수집하고 막힌 문을 통과하면 게임 종료
